# Chrome Web Store Assets

Place Chrome Web Store listing assets in this folder.

Required assets:

- `screenshot-1.png`: 1280x800 PNG or JPEG
- `small-promo-tile.png`: 440x280 PNG or JPEG

Optional assets:

- `marquee-promo-tile.png`: 1400x560 PNG or JPEG

These files are for Chrome Web Store upload only. They are not bundled into the extension package.
